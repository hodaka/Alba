# 地理と産業

## 世界の下敷き — 近畿地方

ティアファは**近畿地方を模した箱庭世界**である。
（→ [地名と民族.md](地名と民族.md) の「摂津／大阪／梅田」の喩えは、単なる比喩ではなく
この下敷きに由来する。）

- そのため**水資源も基本は豊富**。乾きに苦しむ土地ではない。

## 南北の分割

世界は地理的に**南北に分かれている**。
（民族・呼称の対応は → [地名と民族.md](地名と民族.md)：北部＝フスク、南部＝ペルキ。
南北の中間に緩衝地帯アルバ。ただしアルバの成立は楽園ティアファ時代より後
→ [世界の成り立ち.md](世界の成り立ち.md)）

## カロリーベース — 南部の落花生

- **南部**には**落花生が大量に自生**している。
- この落花生が、**世界全体のカロリーベースを支えている**。
- **無限ピーナッツ湧きどころ**があり、**ノーコストで**ピーナッツが湧き出てくる。
- あまりにも**手間なくカロリーが手に入る**ため、**農業が発達しなかった**。
- 世界に**カロリーが満ちている**こと自体が、平和・繁栄の大きな要因。

## 塩 — 北部の岩塩と、南部の製塩難

塩をめぐっては、南北の事情が対照的：

- **北部**：寒冷でカロリーは**輸入に頼っている**。ただし**岩塩が山ほど取れる**のが強み。
- **南部**：**海はあるが外洋には出られない**。加えて**湿潤な気候で製塩に向かない**。
  無理に海塩を作っても**にがりが多くてまずい**。

## 南北の交易補完

上記の結果、南北はおのずと**補い合う交易関係**になる：

| | 得意（供給） | 苦手（依存） |
|---|---|---|
| **南部** | カロリー（無限ピーナッツ）、**紙**（原料がよく育つ） | 塩（自前ではまずい海塩しか作れない） |
| **北部** | 塩（岩塩が豊富）、**インク材料**（よく取れる） | カロリー（寒冷で不足、輸入頼み） |

→ 南のカロリー ↔ 北の塩を軸に、**紙（南）↔ インク材料（北）** も加わり、
南北の交換が世界経済の背骨になっている。
（印刷文化そのものは [文明と技術.md](文明と技術.md)）

## 通貨と貧富 — 楽園ティアファ期

- **通貨は原型しかない。** 貨幣経済は未発達。
- 取引は**ほとんどが物々交換**。
- その結果、**貧富の差がほとんど無い**。
  （豊かなカロリー・平和・未発達な貨幣が重なり、富が偏らない平等な社会。）
