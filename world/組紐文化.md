# 組紐文化

**男女とも組紐を作り、意中の人に渡す**文化がある。求愛・贈り物としての組紐。
（フリーセックス文化・和合の価値観とも地続き → [世界の成り立ち.md](世界の成り立ち.md)）

## 誕生の装身具「サオジ」との関係

組紐は求愛だけでなく、**子の誕生**にも用いられる：
**両親が編んだ組紐に銀のティム管を12本通し、生まれた子に掛けてあげる**——これが「サオジ」。
楽園期には通貨ではなく装身具だった。→ [通貨.md](通貨.md)

## 刺繍は男の手仕事（通念）

- **ティアファ世界では、刺繍は男がするもの。**
  ＝糸撚り・組み紐編み・機織りと並ぶ、**家に縛られた男の屋内仕事**の系譜
  （→ [通貨.md](通貨.md)「男性を家に縛り付ける構造」／[楽園の崩壊.md](楽園の崩壊.md) 掟「男は屋内で組紐を編む」）。
- ゆえに**女が刺繍をするのは、通念から外れた振る舞い**になる
  （実例＝**クラウディ・クラウディア**。少女趣味の乙女ゆえ自分のドレスに刺繍をしていたが、
  洋菓子店の客に難じられて以来やめている → [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)）。
- **ただし小さい子どもは例外**：**子どもが刺繍をしていても性差は問われない**（＝この通念が効くのは
  大人になってから）。実例＝クラウディアの孫**トゥインクル・クラウディア（女の子）**が、祖母のドレスに
  季節の花を刺している。
- **強さは南北で違う**：**北部（フスク）は南部より男女差がそこまで大きくない**ため、この通念も**北では緩い**
  （→ [地名と民族.md](地名と民族.md)「男女差の強さも南北で違う」）。
  ただし**緩い＝無い、ではない**——北部フスクスの首都にいるクラウディアが現に難じられている。
  ＝**南部だったら、もっときつく言われた**ということでもある。
- ※要確認：刺繍と組紐（男女ともに編み、贈り合う → 上記）の**担い手の非対称**。
  組紐は男女ともなのに刺繍は男だけ、という線引きの由来。

## 南北の自然な差

この組紐文化は、**南北で自然と差が出る**（素材・環境に由来する違い）。
※ 公民館（教会）文化が南北で差が無かったのとは対照的
→ [文明と技術.md](文明と技術.md)

| | 北部（フスク） | 南部（ペルキ） |
|---|---|---|
| 素材・染 | **染料が手に入りやすい** | 染めはおとなしめ |
| 質感 | **ザイルのようにカラフルで力強い** | **細くしなやか** |
| 技巧 | （力強い作り） | **高度な編み方・結び方が発達** |

- **北部**：染料が豊富 → 色鮮やかで、ザイル（登山ロープ）のような太く力強い組紐。
- **南部**：細くしなやかな作り。染は控えめだが、**編み方・結び方の技巧が高度**に発達。
