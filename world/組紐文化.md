# 組紐文化

**男女とも組紐を作り、意中の人に渡す**文化がある。求愛・贈り物としての組紐。
（フリーセックス文化・和合の価値観とも地続き → [世界の成り立ち.md](世界の成り立ち.md)）

## 誕生の装身具「サオジ」との関係

組紐は求愛だけでなく、**子の誕生**にも用いられる：
**両親が編んだ組紐に銀のティム管を12本通し、生まれた子に掛けてあげる**——これが「サオジ」。
楽園期には通貨ではなく装身具だった。→ [通貨.md](通貨.md)

## 崩壊後の変質 — 女はサオジを外し、男が編む担い手に

楽園期は男女ともに編み、誰もがサオジを掛けた（上記／[通貨.md](通貨.md)）。だが崩壊後の神話（掟）で、
この担い手と意味が変質する：

- **ガイヤが自らのサオジを外し、その組紐でサオージを縛った**——という神話（→ [楽園の崩壊.md](楽園の崩壊.md)
  「掟の誕生」）ゆえに、**女はサオジを着けず**、**男は失われたガイヤの組紐を編み続ける**務めを負う。
  ＝**組紐を編む担い手が男に固定**される（後の内職「男は屋内で組紐を編む」→ [通貨.md](通貨.md)）。
- **ただし南北差がある**：これは**南部の傾向**で、**北・辺境では女性も普通にサオジを着ける**
  （楽園期の名残 → [楽園の崩壊.md](楽園の崩壊.md)「北・辺境では女もサオジを着ける」）。

## 南北の自然な差

この組紐文化は、**南北で自然と差が出る**（素材・環境に由来する違い）。
※ 公民館（教会）文化が南北で差が無かったのとは対照的
→ [文明と技術.md](文明と技術.md)

| | 北部（フスク） | 南部（ペルキ） |
|---|---|---|
| 素材・染 | **染料が手に入りやすい** | 染めはおとなしめ |
| 質感 | **ザイルのようにカラフルで力強い** | **細くしなやか** |
| 技巧 | （力強い作り） | **高度な編み方・結び方が発達** |

- **北部**：染料が豊富 → 色鮮やかで、ザイル（登山ロープ）のような太く力強い組紐。
- **南部**：細くしなやかな作り。染は控えめだが、**編み方・結び方の技巧が高度**に発達。
