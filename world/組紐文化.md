# 組紐文化

**男女とも組紐を作り、意中の人に渡す**文化がある。求愛・贈り物としての組紐。
（フリーセックス文化・和合の価値観とも地続き → [世界の成り立ち.md](世界の成り立ち.md)）

## 南北の自然な差

この組紐文化は、**南北で自然と差が出る**（素材・環境に由来する違い）。
※ 公民館（教会）文化が南北で差が無かったのとは対照的
→ [文明と技術.md](文明と技術.md)

| | 北部（フスク） | 南部（ペルキ） |
|---|---|---|
| 素材・染 | **染料が手に入りやすい** | 染めはおとなしめ |
| 質感 | **ザイルのようにカラフルで力強い** | **細くしなやか** |
| 技巧 | （力強い作り） | **高度な編み方・結び方が発達** |

- **北部**：染料が豊富 → 色鮮やかで、ザイル（登山ロープ）のような太く力強い組紐。
- **南部**：細くしなやかな作り。染は控えめだが、**編み方・結び方の技巧が高度**に発達。
