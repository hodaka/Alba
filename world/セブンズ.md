# セブンズ

**10人の魔女で構成される集まり（集団）の名前。**

## 名と実の乖離 — 「セブンズ」なのに10人

- 名は「**セブンズ**（Sevens／7）」だが、**実際の構成人数は10人**。
- ＝**名（7）と実（10）が食い違う**。この世界の頻出テーマ（見かけと実情の乖離）の一例。
  **なぜ「セブンズ」で10人なのか（名の由来）は要確認。**

## メンバー

判明しているメンバー（＝これまで名の挙がった主要な魔女）：

- **ベルタ** … 生得＝意識の転移（ハズレ）。“万能の魔女”の看板／実像は人間的才能
  （→ [characters/ベルタ.md](../characters/ベルタ.md)）。
- **クラリス** … 複数の生得能力＋後天取得の両方を持つ、魔法的に規格外の存在。クラリス派の頭目
  （→ [characters/クラリス.md](../characters/クラリス.md)）。
- **B（チェリー・ブロッサム・チェリー）** … 生得＝「応援」（当たり）
  （→ [characters/チェリー・ブロッサム・チェリー.md](../characters/チェリー・ブロッサム・チェリー.md)）。
- **ブル・ベルコーサム** … 生得＝「因果律操作」（当たり）
  （→ [characters/ブル・ベルコーサム.md](../characters/ブル・ベルコーサム.md)）。

## メンバーの性質（判明していること）

- **「当たり生得持ち」の縛りではない。** ベルタ（生得＝意識の転移＝ハズレ）もセブンズゆえ、
  セブンズの選抜基準は“当たりの生得能力”ではない（→ 基準は要確認）。
- **ネクロマンサーはセブンズにいない**（この世界にネクロマンサーは存在するが、セブンズ内に
  該当者なし → [魔法.md](魔法.md)）。

## 要確認

- 名「セブンズ（7）」なのに10人である由来。
- 残り6人のメンバー（＋エルゼ／クラウディア がセブンズか否か）。
- セブンズという集まりの**目的・成立・時期・地位**（何のための集団か）。
- メンバーの選抜基準（“当たり生得”ではない → 何で選ばれるのか）。
