# セブンズ

**10人の魔女で構成される集まり（集団）の名前。実体は——お茶会。**

## 正体 — 不死に飽きた魔女たちのお茶会（確定）

- **時期：本編からさらに数百年後（＝「最終的」な時点）。**
- **不死の魔女たちが、不死に飽き始める。** そして**アルバに集まって、仲良くお茶会をするようになる**。
- **「セブンズ」とは、そのお茶会のメンバーの名前**である。
  ＝物々しい結社でも同盟でもなく、**永すぎる生を持て余した者たちの茶飲み友達の集まり**。
- **不死の仕様への手がかり**：数百年の果てに**「飽きる」**という消耗が来る（→ [魔法.md](魔法.md)「魔女への神罰」。
  不死は専用ファイル化が要るレベルの特殊仕様＝仕様そのものは語り待ち）。

## 名の由来 — 「7」は魔法で罰せられた者の頭数（確定）

- **命名者は不明**（誰が言い出したか分からないまま定着した）。
- **由来＝ベルタ・グラトニーとクラウディ・クラウディアを除いた、「魔法を使っていて罰せられた」者が7人だったから**。
  ＝だから「セブンズ」。
- **ベルタとクラウディアの反応**：
  > 『そうね。私たちはセブンズね。』
  - 二人が**ハモって**そう言い、**ちょっとだけ拗ねた**。
- **それでも「響きがいいから」、会の名前として採用された。** ＝拗ねられたまま押し通った命名。
- ＝ゆえに**「7」という数そのものに大した意味はない**（数秘的な象徴ではなく、たまたまの頭数＋語感で決まった名）。

### ここで解けたこと

- **カウント外の理由（解決）**：ベルタとクラウディアが「7」に数えられないのは、
  **「魔法を使っていて罰せられた」7人に該当しないから**。
  - **クラウディア**＝**魔力ゼロ**。魔法を一度も使っていないのに、目立ちの一点で罰された（＝冤罪）
    → [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)。
  - **ベルタ**＝**ガイヤが罰しに来た時点で、魔法を使っていなかった**（**無神論者で神の声を信じておらず**、
    アナウンスを真に受けなかった）。ゆえに7に入らない。
    ただし**その時点で既にキルカウント二桁**（間接的にではあるが）＝**罰せられるべき人物ではあった。
    少なくとも魔法は使っていない、というだけ** → [characters/ベルタ.md](../characters/ベルタ.md)。
  - ＝**カウント外の2人は「魔法を使わずに罰せられた」で括られる**が、中身は正反対
    （**クラウディア＝完全な無実**／**ベルタ＝咎はあるが、罰の名目とは別の咎**）。
- **プレッサ**＝**後から加入**ゆえ原初の7の外（加入の時期・経緯は要確認）。
- ＝ 7（魔法で罰せられた者）＋2（カウント外＝ベルタ・クラウディア）＋1（後加入＝プレッサ）＝ **10人**。
- ＝**名（7）と実（10）が食い違う**、この世界の頻出テーマ（見かけと実情の乖離）の一例でもある。
  ただし本件は**謎ではなく由来の分かっているカラクリ**（＝上記）。

  該当者なし → [魔法.md](魔法.md)）。

## 「7」という数に、大した意味はない（確定）

- **セブンズは重要なキーワード**（集まりそのものは世界の要）。**ただし「セブンズ」の“数”＝7 に大した意味はない**（作者言）。
- **由来は判明している**（＝魔法を使って罰せられた者の頭数＋響きの良さ → 上記「名の由来」）が、
  それは**たまたまの頭数と語感**であって、**数秘的な象徴・必然性はない**。そこを詮索しにいかないこと。
