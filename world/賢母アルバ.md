# 賢母アルバ（けんぼアルバ）／マッドアルバ

始祖アルバ（白狼 → [始祖アルバ.md](始祖アルバ.md)）の火刑から **百余年後** に現れる人物。
**「アルバ」の名を冠される、二人目の傑物**（＝アルバの歴史をふたたび動かす者）。

## 二つの名 — 振れ幅

- **賢母アルバ**（後世の敬称）… 賢い母。敬い讃える側の名。
- **またの名：オナニー狂いの天才錬金術師マッドアルバ** … 奇矯・罵倒の側の名。
  - **天才錬金術師** … この世界に **錬金術（とその天才）** が現れる。
  - **マッドアルバ** … 狂気・奇人の含み。
  - **「オナニー狂い」** … 性的に自足する／種馬経済に依らぬ奇矯、の含み（強いワード。
    ※在世の **通り名（またの名）** として記録。cf. プーカ「売女」等の罵倒系通り名）。

> ＝**賢母（聖）** と **マッド／オナニー狂い（俗・奇）** が同居する、**振れ幅の大きい通り名**。
> プーカ（天眼／売女 → [プーカ.md](プーカ.md)）・始祖アルバ（始祖アルバ／ベル → [始祖アルバ.md](始祖アルバ.md)）と
> 同じ、この世界の傑物に共通する「評価の割れ」。

## 現れた時代 — 大戦の記憶が薄れ、病と貧富は続く

賢母アルバが現れるのは、次のような世相：

- **大南進（大戦）から百余年**。**大戦の記憶は薄まりつつある**（伝説化の方向 → cf. 青の衝撃の伝説化
  [楽園の崩壊.md](楽園の崩壊.md)）。
- **青死病（碧斑熱）は、相変わらず猛威を振るう**（＝風土病として消えず続く → [楽園の崩壊.md](楽園の崩壊.md)）。
- **貧富の差は拡大**し、**「和合」の価値観はさらに遠のいている**（楽園ティアファの美徳が、時代とともに
  薄れ続ける → [世界の成り立ち.md](世界の成り立ち.md)／[大南進.md](大南進.md) 死者の票による貧富の固定化とも地続き）。

> ＝始祖アルバの殉教から百余年、世界はなお病み、貧富に裂かれ、古い和合は遠い。
> その時代に、天才錬金術師「賢母アルバ／マッドアルバ」が現れる。（何を為すかは今後の語りで。）

## アルバ神学での扱い — 解釈が割れる「ヤバいやつ」

- **アルバ神学**（＝アルバ独自の神学。アルバの教国化④・カルト化と地続き
  → [地名と民族.md](地名と民族.md)「大戦後の宗教地図」）の中でさえ、
  **賢母アルバをどう扱うか、解釈の論争がある**。
- ＝賢母（聖女）として祀りたい一方、その奇矯・異端性（マッド／オナニー狂い／錬金術）ゆえに、
  神学的な位置づけが定まらない。**それほどに“ヤバいやつ”**。

> 対比：始祖アルバは **南の新約から異端** 扱い（→ [始祖アルバ.md](始祖アルバ.md)）。賢母アルバは
> **アルバ自身の神学の内で論争の的**。二人のアルバは、どちらも宗教にとって **綺麗に収まらない** 存在
> ——聖性と逸脱が同居する。
