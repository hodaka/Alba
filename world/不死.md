# 不死

**ガイヤの神罰の内容のひとつ。そして「魔女」の定義そのもの**
（魔女＝不死を持つ者＝ガイヤに罰せられた者 → [魔法.md](魔法.md)「魔女の定義」「魔女への神罰」）。

> **⚠この世界の不死はかなり特殊で、独立した項目で扱うレベルの仕組みを持つ**（作者言）。
> このファイルは**語られた分だけを記録する器**。**仕様は語り待ち＝書記は勝手に作らない。**

## 判明していること

### 不老 — 前提

- **不老が前提。** 不死者は**歳を取らない＝姿がその時点で止まる**。
- ※ガイヤ自身が**魔女と同じ若い見た目のまま**である（→ [世界の成り立ち.md](世界の成り立ち.md)「ガイヤの姿・質」）
  のとも地続き。

### 死の判定とリスポーン

**不死＝「死なない」ではない。「死んでも戻ってくる」。**

1. **判定は生前の耐久値で行われる。** ＝不死者になっても**肉体が頑丈になるわけではない**。
   その個体が生前に持っていた耐久のまま、通常どおり「死んだ」と判定される。
2. **本人は死を体験する。** 判定されたら、まず**実際に死ぬ**。死は回避も省略もされない。
3. **スティグマから肉体が再構成される。** そこから復帰する＝これが**リスポーン**。

- ＝**聖痕（スティグマ）はリスポーンの起点**。神罰の二本立て（スティグマ＋不死 → [魔法.md](魔法.md)
  「魔女への神罰」）は独立した二つの罰ではなく、**スティグマが不死の器・拠点として機能している**ことになる。
- **派生：耐久が生前のままということは、頑丈な魔女は元から頑丈**。
  クラウディアが核融合のゼロ距離直撃で死ななかったのは不死の効果ではなく、
  **純粋にその肉体の耐久値がそれを上回っていた**という意味になる
  （→ [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)）。
- **詳細は語り待ち**（回数の制限、再構成にかかる時間、失われるもの／保たれるもの、記憶や所持品の扱いなど、いずれも未確認）。

#### 【メタ】スティグマを持たない者 — クラウディアの場合（解決）

**問い**：リスポーンの起点がスティグマなら、**スティグマが不発に終わったクラウディ・クラウディア**
（上腕二頭筋・筋肉で弾いて霧散 → [魔法.md](魔法.md)）が死んだらどうなるのか。

**答え＝作者にも分からない。そして、その問いは立たない。**

- **彼女には別次元の加護がある**（作者の比喩＝**『こち亀』の両津勘吉のような不死身さ**）。
  ＝**リスポーンが起きるような事態そのものが訪れない。**
- **⚠ただし「何でもメタで片付く」という話ではない。**
  例えば**スーパー魔女大戦（バトルロイヤル）でベルタの核熱攻撃をギリギリ耐えた一件は、メタ抜きでも説明がつく**——
  彼女は**「消力（シャオリー）」と「三戦（サンチン）」で正面からきちんと防御している**
  （→ [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)「戦い方」）。
  ＝**作中の説明がつくものは、作中の理屈で説明する。** メタは最後の逃げ道であって、第一の説明ではない
  （＝「魔法を万能の逃げ道にしない」方針と同じ構え → [../CLAUDE.md](../CLAUDE.md)）。
- ＝これは**作中の法則ではなくメタの扱い**（[キルカウント.md](キルカウント.md) と同じく、作者の側にある取り決め）。
- **⚠ただし、シリアスモードではダメージは通る。** ＝無敵キャラではない。
  **作品のトーンによって彼女の耐久は変わる**——ギャグ回では何をされても「いってー」で済み、
  真面目な場面では普通に傷つく。
- **書記の運用**：①**作中の因果として持ち出さない**（「彼女は別次元の加護があるから死なない」と地の文や
  設定で説明しない）。②**リスポーン不能を悲劇として演出しない**（＝前に書いた「唯一、本当に死ねる個体」の
  読みは採らない）。③**ギャグ回の耐久を、そのままシリアスな場面の数値として一般化しない。**
- ※**🔒サオージの加護とは別物**。サオージの加護は作中の設定（魔力ゼロの原因）であり、
  神罰を弾くような力はないと本人が明言している（→ [魔法.md](魔法.md)）。こちらはメタの側の話。
- **実例（回数）**：**クラウディ・クラウディアは、一度もリスポーンしていない**
  （→ [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)）。
  ＝**不死者になってから今日まで、一度も死んでいない**ということになる。
  - ※**魔女は全員が全員と一度以上ガチで殺し合っている**（→ 下記）ことを踏まえると、
    **彼女はその全部をくぐって、一度もリスポーンしていない**ことになる。
    ただし**リスポーンが「死んだとき」に起きるものかは未確定**のため、断定はしない。

### 生殖能力 — 不妊／「永遠の処女」

- **正確には「子を成せない」ではなく「不妊」。** 行為が禁じられるのでも避けられるのでもなく、
  **孕まない**。＝死なない代わりに、次の生命も生まれない（→ [魔法.md](魔法.md)「魔女への神罰」）。
- **この世界観では、それは「永遠の処女」になるということ**（作者言）。
  - ※**和合の文化**（→ 下記「魔女どうしの関係」／[世界の成り立ち.md](世界の成り立ち.md)）を持つこの世界で、
    子を成すことのない身体を持ち続ける、という意味。**この語の正確な射程は要確認**
    （不妊の言い換えか、リスポーンによる肉体再構成の帰結か → 下記「要確認」）。
- ※例外的に、**ガイヤからの「補償」として与えられた家族**の例がある
  （サニー・トゥインクル → [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)。
  経緯は [stories/](../stories/) に本文が入る予定）。

### 長すぎる時間の帰結 — 魔女どうしの関係

**魔女は暇を持て余している。** その結果：

- **全員が全員と、一度以上の肉体関係がある。**
- **全員が全員と、一度以上ガチで殺し合っている。**
- ＝**魔女という集団は、互いに寝た相手であり、殺し合った相手でもある**という関係で編まれている。
  数百年という時間が、あらゆる組み合わせを一度ずつ通してしまった形。
- ※**肉体関係のほう**は、この世界の**和合の文化**（旧約ティアファの唯一の教義「和合をもって、地に満ちなさい」／
  ティムを交換し合うフリーセックス文化 → [世界の成り立ち.md](世界の成り立ち.md)・[通貨.md](通貨.md)）とも地続きで、
  背徳の話ではない。
#### 決着のつけ方

- **必ずしも死で決着しない。**
  - **飽きて終わる**こともある。
  - **事前にルールを決める**こともある（例：**リスポーンしたら終わり**）。
- ＝**「負けた」＝「死んだ」ではない。** 負けても一度もリスポーンしていない者もいる
  （クラウディ・クラウディア。**無敗ではないが、リスポーンは一度もなし**
  → [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)）。

#### 暇の産物 — 魔女の「天下一武道会」

- **暇を持て余した魔女たちが、天下一武道会のようなものを開催したことがある。**
- **優勝はB（チェリー・ブロッサム・チェリー）。**
- **⚠ただし、なぜかB以外の全員が会場までたどり着けなかった**＝**Bは一戦も戦わずに優勝した**
  （＝Bの生得能力「応援」＝万物のえこひいきの発露
  → [characters/チェリー・ブロッサム・チェリー.md](../characters/チェリー・ブロッサム・チェリー.md)「タイマン全勝の正体」）。
- ＝**Bの「タイマン全勝」という戦績の出所はこれ**。**勝ち続けた記録ではなく、不戦勝の山。**
- ※**名称・開催時期・参加者の範囲（セブンズだけか、魔女全体か）は未確認**
  （→ [canon-notes.md](../canon-notes.md)）。**書記は作らない。**

#### キルカウントには入らない

- **魔女は、どれだけリスポーンさせてもキルカウントには含まれない**
  （→ [キルカウント.md](キルカウント.md)）。
  ＝**不死者を「殺した」とは数えない。** 何度やり合っても、帳簿は動かない。

- **要確認**：**セブンズ**（→ [セブンズ.md](セブンズ.md)）が**仲良くお茶を飲めている**のは、この全員経験済みの果てに
  落ち着いた状態、ということになるのか。

### 時間の使い方（例）

- 無限の時間を何に使うかは、魔女によって違う。
- **例：クラウディ・クラウディア** … **不死者になっても武術の訓練なんてしたくない。焼き菓子を焼いている方がいい**
  （→ [characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)「戦い方」）。
  ＝**数百年あっても、武術を身につけることは選ばなかった**。
  **ただし筋トレはする**（**もう魂に刻まれている**から＝強くなる手段ではなく、すでに習慣そのもの）。
  ＝**焼き菓子を焼き、ダンベルを上げる数百年**。持て余した時間の埋め方が、そのまま人物を表す。

### 消耗 — 「飽きる」

- **数百年の果てに「不死に飽きる」**という消耗が来る。**セブンズ**（＝不死に飽きた魔女たちのお茶会）は
  その帰結（→ [セブンズ.md](セブンズ.md)）。

## 要確認（未着手の中核）

- **「永遠の処女」の射程**：①不妊の言い換え（子を成す側になれない）か、
  ②**リスポーンで肉体が再構成される＝物理的に元へ戻る**のか。②なら「魔女は全員が全員と肉体関係がある」
  （下記）との関係が変わる。
- **リスポーンの仕様の残り**（回数・代償・所要時間／記憶・所持品・その場の位置はどうなるのか）。
- 死の判定が「生前の耐久値」なら、**老衰・病死・自死はどう扱われるのか**（不老ゆえ老衰は来ないはずだが、
  クラウディアの夫の中風のような病はどうか）。
  - ※**ベルタの大気圏再突入生還は「基本系統の肉体強化の力技」と説明済み**＝**死んでいない**側の説明だが、
    不死とどちらが効いたのか（→ [魔法.md](魔法.md)「未確定」）。
- 教祖アルバの改竄正史が掲げる「不死」モチーフとの関係（→ [canon-notes.md](../canon-notes.md)）。
- 不死を**罰以外の経路**で得た者がいるのか（サニー・トゥインクル → [魔法.md](魔法.md)「定義の縁が揺れる保留」）。
