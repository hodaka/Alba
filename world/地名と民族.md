# 地名と民族 — 呼称の由来

## 三層の呼称（北部・南部）

| 区分 | 北部 | 南部 |
|------|------|------|
| 地方名（現在語・短い形） | フスク地方 | ペルキ地方 |
| 大きな国 | フスクス軍事同盟 | ティアファ教国（通称ペルルキ） |
| 民族名 | フスク人 | ペルキ人 |

## 民族分化の起源（青死病の崩壊期）

フスク人／ペルキ人という**民族アイデンティティは、楽園期には存在しなかった**。
青死病パンデミック後、男性の扱いが南北で真逆になったこと（北＝働く男／南＝囲われる男）を
基礎に、互いを別の民族と意識するようになった。→ 詳細 [楽園の崩壊.md](楽園の崩壊.md)

- **フスク人（北）**：無骨なマッチョ文化。
- **ペルキ人（南）**：線が細く華麗な文化。

## 由来

もともと北部・南部は古語で「フスクス」「ペルルキ」と呼ばれていた。
のちにそれぞれの地に大きな国が成立し、地方名の方は短く変化して
「フスク地方」「ペルキ地方」と呼ばれるようになった。

- **北部の大国**：フスクス軍事同盟。
- **南部の大国**：ティアファ教国（正式名）。通称「ペルルキ」。**奉じるのは新約ティアファ**
  → 下記「大戦後の宗教地図」。
- 「フスクス」「ペルルキ」＝古語であり、北部では大国（軍事同盟）の名、南部では大国の通称でもある。
- 「フスク」「ペルキ」＝現在語の地方名。民族名（フスク人／ペルキ人）もこの短い形をとる。

## 呼称の空気感（重要）

住人は「古語か現在語か」を**意識していない**。音が近いため、両者を明確に区別せず混在して使う。
（現実で言えば「摂津／大阪／梅田」のような関係。指すものはおおむね同じ圏で、
どれで呼ぶかは話者の癖や場面まかせ。厳密な使い分けの規則は無い。）

→ セリフや地の文では、フスクス／フスク（ペルルキ／ペルキ）どちらが出てきても正しい。
本文で古語・現在語が混在していても誤りではない。

※ ただし「◯◯人」という民族名は短い形が据わりがよい（関西人はOK・梅田人は違和感、の感覚）。
民族名は **フスク人／ペルキ人** を正とする。

## 中間地帯

- **アルバ**：南北の中間に存在する緩衝地帯。この地方独自の白い植物が自生する。

### 場所としてのアルバの特徴（楽園期から存在）

**場所としてのアルバは、楽園ティアファ期からすでに存在する**（← 政治的な「緩衝地帯」の
成立とは別。土地そのものは昔からある）。便利な土地として知られる：

- **固有種の真っ白な蔦性植物**が、たくさん生えている（＝「白い植物」の正体）。
- **清水と温泉が、中央のボトルネックの小高い丘で湧く**（＝アルバの心臓部・人が住む中核）。
  この清水は**枯れない無限水源**。地理的には**東の高山に源を発する**と見るのが無難だが、
  **由来そのものは重要でない**——要点は「**アルバが兵糧攻め（包囲）に遭っても水が尽きない**」
  という**防御装置**である点。（世界の「無限◯◯湧きどころ」の一種 →
  cf. 無限ピーナッツ [地理と産業.md](地理と産業.md)）
- 温泉は本編「白い湯の秘密」の温泉描写とも整合。
- 地形の帯構成（西＝湿地／中央＝丘・水と温泉／東＝高山＝事実上の壁）の詳細は
  → [地理と産業.md](地理と産業.md)。

## 「アルバ」という語 — 意図的な多義語（トラッカー）

⚠ **「アルバ」は今後いろいろな意味で登場する。作者が意図的にややこしくしている**多義語。
（ティアファが「世界／宗教／国家／文化」の顔を持つのと同種の仕掛け。）
アルバが出てきたら、**どの意味のアルバか**を必ず文脈で見極めること。

現時点で判明している意味：

1. **場所（地名）としてのアルバ** … 南北をつなぐ砂時計状のボトルネックの土地。交通の要衝。
   西＝湿地／東＝山。→ [地理と産業.md](地理と産業.md)、本編「白い湯の秘密」の舞台。
2. **緩衝地帯としてのアルバ** … 南北の中間の緩衝地帯（上記1と重なる可能性大だが、
   政治的な「緩衝地帯」という含みでの用法）。白い植物が自生。
   **成立時期・経緯（確定）**：青の衝撃後の混乱期、南北の狭間（西側湿地を中心とする一帯）が、
   **南北双方が領有を主張しつつ、実質どちらも統治できない貧困地帯**として立ち上がった。
   ＝これが緩衝地帯②の成立 → [楽園の崩壊.md](楽園の崩壊.md)。
   （貧困は主に西の湿地に偏る。中央の丘＝水・温泉を持つ心臓部は資源を保つ。）
3. **独立勢力（小国）としてのアルバ** … この時期、アルバは南北どちらにも属さない
   **独立した小国**として立つ。中央の高地（＝小高い丘）を本拠に、西の湿地を海（湾）側まで
   牽制して独立を保つ。
   - **係争地は主に西側の湿地帯**（南北双方が領有権を主張／だが双方とも統治しきれない）。
   - **アルバ本体は別格**：**南北双方が欲しがるが、双方ともアルバに一目置き、
     アルバはどちらの陣営にも与しない**（中立の独立）。
   アルバ本体は貧困地帯ではなく、交易で栄える自立勢力。現実の参照＝**堺（自治都市）**。
   地形マッピング → [地理と産業.md](地理と産業.md)。（※あくまで「この時」の状態。時代が下れば変わりうる。）
   - **大南進（停戦）後**：アルバの独立は失われ、**北の旗が立ち北の衛兵が置かれる一方、
     運営資金と運営は南が担う**という「名（北）と実（南）の分離」状態になる
     （＝本文「戦乱の度に主が変わる」の最初の一例 → [大南進.md](大南進.md)）。
   - **その後（長期・二次大戦なき時代）**：この名実分離が続くことで、アルバは逆に
     **不可侵の独立勢力**へと変貌し、南北の**調停地**になっていく。＝失った名目上の独立が、
     より強い「不可侵の独立」へ反転する → [大南進.md](大南進.md)「戦後の長い均衡」。
4. **ティアファ教の教国としてのアルバ（到達点・未成立）** … アルバは最終的に
   **ティアファ教を奉じる「教国」として成立する**（＝南部ペルルキと同じ「教国」の顔を、
   アルバも持つに至る）。ただし**作品時間内でかなりの年月がかかる**成立過程で、一足飛びではない。
   現在地（大戦直後）はまだその手前——無印「ティアファ」を掲げ、後述の保守化・カルト化を経ていく段階。
   → 下記「大戦後の宗教地図」。
5. **人名（称号）としてのアルバ — 「始祖アルバ」** … 大南進開戦直後、北部軍事同盟のアルバ攻略
   最前線にいた女性将校「白狼」が、**正史では「始祖アルバ」と記される**（＝アルバ〔国〕の創始者）。
   ＝「アルバ」は**人の名（称号）**の顔も持つに至る → [始祖アルバ.md](始祖アルバ.md)。
   （民間伝承では姓＝「ベル」。以後、地の文で「アルバ」が**人**を指す用法が出うる。）
   - **反復する称号**：「アルバ」は一人に留まらない。始祖アルバの百余年後に **賢母アルバ（またの名
     マッドアルバ）** → [賢母アルバ.md](賢母アルバ.md)、そのさらに十数年後に **教祖アルバ**
     → [教祖アルバ.md](教祖アルバ.md)。＝「アルバ」は**アルバの歴史を動かす傑物（いずれも女性）に
     冠される名（称号）**の性格を帯びる。現時点で三代：**始祖アルバ（白狼）／賢母アルバ（マッドアルバ）／
     教祖アルバ**（要確認：命名／襲名の規則）。

（以降、別の意味のアルバが出るたびにここへ追記する。）

> 時間軸メモ（解決）：「楽園ティアファ時代にはアルバはまだ無い」（→ [世界の成り立ち.md](世界の成り立ち.md)）
> の“アルバ”は **②緩衝地帯（政治的意味）** を指す。**①場所としてのアルバは楽園期から存在**する。
> ＝土地は昔からあるが、南北の緩衝地帯としての性格を帯びるのは後。

## 大戦後の宗教地図 — 新約ティアファと諸勢力

大戦（大南進）後、ティアファ教は**新約**を軸に地図が引き直される。旧約（楽園期の原本・
教義は「和合」一つ → [世界の成り立ち.md](世界の成り立ち.md)）に対し、**新約ティアファ**は崩壊期に
生まれた「掟」の系統（男の原罪・組紐の捧げ・女性上位 → [楽園の崩壊.md](楽園の崩壊.md)）と見られる。

- **ペルルキ（南部＝ティアファ教国）＝新約の本流。** そして**かなり大衆化した**——大国ゆえ、
  新約はメインストリームの宗教として広く薄く社会に浸透する。
- **アルバ＝新旧どちらの看板も上げない。** ただ無印の**「ティアファ」とだけ宣言する**
  （＝南北どちらの宗派にも与しない中立の姿勢。緩衝地帯・独立勢力としてのアルバと一貫
  → 多義語トラッカー②③）。
  - **だが実態はかなり新約寄り**（本音では新約に近い）。
  - **ペルルキ勃興後、アルバは保守化した。** 新約の重量級ペルルキが強大化すると、アルバは
    逆に距離を取り、相対的に保守（旧約寄り）へ振れて独自の中間ポジションを保った。
  - ＝結果、アルバの信仰は**ややカルト化**する（小さく閉じて濃くなる）。
    大国ペルルキが大衆化＝薄まるのと**真逆**（規模と宗教の濃度が反比例）。
  - アルバはやがて **独自の神学（アルバ神学）** を持つに至る（＝教国④・カルト化の帰結）。その神学は、
    アルバの傑物（始祖アルバ・賢母アルバ・**教祖アルバ**）の**扱いを巡って論争を抱える**
    → [賢母アルバ.md](賢母アルバ.md)／[教祖アルバ.md](教祖アルバ.md)。
    とりわけ **教祖アルバ**（アルバ神学を興した当人）の正体をめぐっては、
    **化身説／神託の巫女説（定説）** の二派に割れる（＋考古学の新興宗教創始者説）→ [教祖アルバ.md](教祖アルバ.md)。

> ⚠ 別立て：**このアルバのカルトは、それだけで物語一本分の規模**がある。ここでは深掘りせず、
> 独立したスレッドとして切り離す（→ [canon-notes.md](../canon-notes.md)）。
> 効き所（要確認）：楽園の崩壊.md が前方参照する「**本編の女性上位主義カルト**」、そして
> **本編「白い湯の秘密」の舞台＝アルバ**——この「アルバのカルト化」が、本編で描かれる
> 女性上位カルトの**源流**につながる線と見られる（断定は保留）。

## 人名の命名規則 — 中間地帯（アルバ）の「◯ー✕」型

**中間地帯（アルバ）では、楽園ティアファの時代から、人名を「◯ー✕」の形にする風習がある。**

- 形＝**2音めが長音（ー）**になる名。頭と尻に音を置き、あいだを「ー」で伸ばす。
  - 例：**ノーマ**（ノ・ー・マ）／**マーナ**（マ・ー・ナ）。
- ＝この形の名を持つ者は、**中間地帯（アルバ）の出身**であることを示す。
- 中間地帯には**名家（由緒ある家）が多い**（楽園期の交易の富に由来 → [地理と産業.md](地理と産業.md)）。
  ノーマ・マーナもそうした名家の出。

> 本文からの裏づけ：外伝「白い湯の秘密」の登場人物 **マーナ（本名 マーナ・サティウス）** と
> **ノーマ** は、いずれもこの「◯ー✕」型（→ stories/アルバ　外伝　白い湯の秘密.txt）。
> ＝命名規則は本文と整合。名は「◯ー✕型（マーナ）＋姓（サティウス）」の構成と見られる。
