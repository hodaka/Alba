# 始祖アルバ（白狼）

大南進（初の世界大戦）開戦直後、**北部軍事同盟のアルバ攻略最前線にいた女性将校「白狼」**。
のちに **正史で「始祖アルバ」と記される**、アルバ（国）の創始者。
＝北の英雄「白狼」（→ [白狼.md](白狼.md) の称号）を担う一人であり、canon-notes の
「もう一人の白狼」スレッドの本体。

## 名の二層 — 正史「始祖アルバ」／民間伝承「ベル」

- **正史**：白狼は **「始祖アルバ」** と記載される（＝アルバという国の始祖）。
  - ＝ここで「**アルバ**」は**人の名（称号）**の顔を持つに至る
    → [地名と民族.md](地名と民族.md) 多義語トラッカー⑤。
- **民間伝承**：彼女の **ファミリーネーム（姓）は「ベル」** とされる。
  - **「ベル」＝フスクス（古語・北）で「北の寒い方」の意味。**
  - ＝姓は正史では失われ、民間伝承にのみ「ベル」が残る
    （姓ロストの motif ＝初代白狼・プーカも同様 → [白狼.md](白狼.md)／[プーカ.md](プーカ.md)）。

## 「異端」— 南の新約から見た始祖

- 「ベル＝北の寒い方」という **北（フスク）由来の含み** ゆえ、
  **新約ティアファ（＝南部ペルルキの宗教 → [地名と民族.md](地名と民族.md)「大戦後の宗教地図」）では
  異端扱い** される。
- ＝アルバの始祖は、もとは **北の兵**（アルバを攻めた側の将校）。その北由来が、南の新約正統から見れば
  受け入れ難く、**アルバの創始者でありながら、南の主流宗教にとっては異端**、というねじれを負う。

> 反転メモ：**アルバを“攻めた”北の一将校が、アルバの“始祖”になる**。アルバが「戦乱の度に主が変わる」
> 土地であること（→ [地理と産業.md](地理と産業.md)）、「名＝北／実＝南」の名実分離（→ [大南進.md](大南進.md)）、
> 無印「ティアファ」を掲げるアルバの中立とも響く、南北が入り混じるアルバらしい出自。

## 人物（既出の確定）

- **女性。** 階級は **中尉** くらい。
- 兵科は **神官騎士**（しんかんきし）＝平たく言えば **衛生兵**。ただし **普通に武器を持って戦う**
  （＝戦闘衛生兵）。
- **戦鎚（せんつい）の達人にして一騎当千。** 武器を持って戦う神官騎士の中でも際立った武勇の持ち主。
  その **武勇を買われ、本人も意気揚々と戦陣を切った**（＝アルバ攻略戦の先陣を、自ら勇んで切り開いた）。
- 役割は、**戦死した仲間のサオジを回収するシンガリ（殿軍）**。
  （サオジ＝通貨化した装身具。戦死者のサオジ回収は経済・儀礼・弔いが一体 → [通貨.md](通貨.md)／
  [楽園の崩壊.md](楽園の崩壊.md)。）

> ⚠ 要確認（先陣↔殿軍）：「意気揚々と戦陣を切った（先陣）」と「サオジを回収するシンガリ（殿軍）」の
> 関係は要確認。(a) 気性は先陣・務めは殿、の両面か／(b) はじめ意気揚々と先陣→のち殿軍でサオジを拾う役へ、
> という時系列（「意気揚々」からの変化）か。順序が確定したらここを更新する。

## 「白狼」（称号）との関係 — 要確認

- 初代白狼は **開戦前に処刑** され、以後「白狼」は北部軍事同盟トップの **称号** でもある
  （征夷大将軍型 → [白狼.md](白狼.md)）。
- 中尉クラスのこの「白狼」が、その称号を **継いだ／継ぐ** ものか、個人の **通り名** か、
  **別軸** かは **要確認**（＝「もう一人の白狼」スレッドの核 → [canon-notes.md](../canon-notes.md)）。
