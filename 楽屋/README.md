# 楽屋 — 非正史。与太話・メタ・初期案の置き場

**ここは正史ではない。** ボツ案・初期設定・楽屋落ち・メタな与太話を放り込んでおく場所。

## 何を置くか

- **Gemini と会話していた頃の初期の設定**（例：**初期のベラ** ＝ 現在のベルタ・グラトニーの前身）。
- ボツになった案、途中で変わった設定、採用しなかった分岐。
- **楽屋（メタ）扱いの発言**——作中の誰も知らない、世界の外側からの言葉。
  - 例：🔒**サオージ談**『えっ ガイヤちゃんのスティグマが弾かれた？　何それ怖い……俺の加護そんな力ない』
    （→ [../characters/クラウディ・クラウディア.md](../characters/クラウディ・クラウディア.md)）
- **メタの取り決め**（作中の法則ではないもの）。
  - 例：**キルカウント**＝作者が処遇を決める物差し（→ [../world/キルカウント.md](../world/キルカウント.md)）／
    **クラウディアの「別次元の加護」**（→ [../world/不死.md](../world/不死.md)）

## 何のためにあるか

**今後のインスピレーションのため。** 面白そうなものを拾って、**これから作るものの種にする。**
＝**過去の記録を保存するためではなく、これから使うために置いてある。**

## ⚠取り扱いのルール（重要）

1. **正史ではない。** ここにある記述は、`stories/` `world/` `characters/` と**同じ重みを持たない。**
   食い違っても、**常に正史（本文 → 二次資料）が勝つ**（→ [../CLAUDE.md](../CLAUDE.md)「最重要原則」）。
2. **⚠【最重要】本編作業中は、このディレクトリを参照禁止。**
   **`stories/` `world/` `characters/` `canon-notes.md` を書く・読む・考えるときに、
   `楽屋/` を開いてはならない。** 参考にするのも、根拠にするのも禁止。
   - **理由**：**読まなければ、混ざりようがない。**
     判断力で防ぐのではなく、**そもそも視界に入れない**ことで防ぐ。
   - ＝**「昔こう書いてあったから」で、現在の設定を動かさない。**
   - **開いてよいのは、語り部が明示的に「楽屋を見よう」と言ったときだけ。**
3. **ここから正史へ引き上げるときは、語り部の裁定が要る。**
   書記の判断で「面白いから採用」としない。**拾うのは語り部、起こすのが書記。**
4. **逆に、正史から降ろされたものをここへ置くのは自由。**
   ボツになった案は捨てずにここへ。

> ＝**楽屋は、読みに来る場所ではなく、置きに来る場所。**
> 使うときは、語り部が「あれ使おか」と言ったときだけ。

## 使い方のヒント

**本編（`stories/`）に詰まって、外側へ出たくなったときの行き先。**
物語の中で行き詰まったまま止まるより、ここへ置きに来るほうがいい
（→ [初期案（Gemini時代）.md](初期案（Gemini時代）.md)「作者の癖」）。
**ただし、置きに来るだけ。本編を書いている最中に読みに来てはいけない**（→ 上記ルール2）。
