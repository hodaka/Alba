# characters/ — 二次資料（キャラ設定）

本文から導かれる二次資料。キャラクターの設定を置く。

ここを編集する時は**必ず `stories/` の本文と突き合わせる**。本文に無い設定を勝手に足さない
（要約時の盛りすぎ・取り違えを防ぐため）。

本文と食い違った場合は、常に本文（stories）が正。

## 記載の約束 — 通り名の欄

- **魔女は全員「〇〇の魔女」という通り名を持つ**（→ [world/魔法.md](../world/魔法.md)「通り名」）。
- 各人物ファイルの冒頭に `- **通り名**：『〇〇の魔女』` の欄を設ける。
- **語られていない人物は `未確認`**（**書記が考えて付けない**）。

## 記載の約束 — キルカウント欄

- **各人物ファイルの冒頭「人物紹介」に、キルカウントの欄を設ける**
  （→ [world/キルカウント.md](../world/キルカウント.md)）。
- 書式：`- **キルカウント**：（値）` ＋必要なら時点や補足。
- **値が語られていない人物は `未確認`** と書く（**推定・計算で埋めない**）。
- ※キルカウントは**作中では誰も認知していない、純粋にメタの設定**。
  資料には明示するが、**作中の因果として扱わない**。
