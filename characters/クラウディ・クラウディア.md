# クラウディ・クラウディア

**正真正銘の魔女**（ただし**魔力ゼロ** → 下記／**魔力の有無は魔女の条件ではない**）。
**セブンズの一員**だが、**「7」のカウント外**（→ [world/セブンズ.md](../world/セブンズ.md)）。

通り名は **『清廉の魔女』**。
本編『白い湯の秘密』（→ [stories](../stories/)）では**物語の幕開けを担う視点人物**。呼び名は主に「クラウディア」
（「クラウディ」が名、「クラウディア」が姓）。

- **通り名**：**『清廉の魔女』**（→ 下記／[world/魔法.md](../world/魔法.md)「通り名」）。
- **キルカウント**：**0**（→ [world/キルカウント.md](../world/キルカウント.md)／下記「キルカウント＝0」）。

## 通り名『清廉の魔女』 — 裏表がない

- **通り名は『清廉の魔女』。**
- **裏表のない人物。** 隠し事をして人を欺くタイプではなく、**そのまま**の人。
- **ただし清廉＝穏当、ではない。** **クラウディアもネジが飛んでいる＝ヤベー奴ではある。**
  その手触りは、彼女がトップを務めた**アマゾネス自警団のモットー**に出ている：
  > **「（殴って）動かなくなってから会話しよう。」**
  （→ [world/アマゾネス自警団.md](../world/アマゾネス自警団.md)）
- **噛み合うこと**：それでいて**キルカウントは 0**（→ 下記）。
  ＝**さんざん殴っておきながら、誰一人殺していない。**「動かなくなるまで」で止めている。
  ＝清廉の通り名は、**綺麗事ではなく、この一線を守り続けたことの側にある**。

### ※重要な書き分け — 「裏表がない」と「見かけと実情のずれ」

- この人物には**見かけと実情のずれが山ほどある**（魔女なのに魔力ゼロ／兄貴分の中身が乙女／
  殺していないのに凶暴視 等 → 下記）。だが**それは本人の作為ではない。**
- **彼女は隠していない。周りが勝手にずれて見ているだけ。**
  ＝**裏表のなさと、乖離の多さは矛盾しない。** 資料でこの人物を書くときは、
  **「偽っている」「使い分けて欺いている」という書き方をしないこと。**
- 一人称の使い分け（対外「俺」／プライベート「私」 → 下記「中身」）も、**欺きではなく役割**として扱う。

## 中身 — バリバリ少女趣味の乙女（重要）

- **本当の中身は、バリバリ少女趣味の乙女。**
- **ガテン系の兄貴分は、対外的にリーダーをしている時の“顔”にすぎない**
  （自警団のトップ時代／街での顔役としての振る舞い）。
- **一人称の使い分け**：
  - **対外・リーダー時 … 「俺」**（豪快な兄貴口調 → 本編『白い湯の秘密』の姿）。
  - **プライベート … 「私」**（素の乙女の口調）。
- **カントリードレスも刺繍も、クラウディア本人の趣味**（娘や孫に着せられているのではなく、
  **本人が好きでそうしている** → 下記「外見」）。
- **乙女っぽいことは、一通り全部大好き。** 憧れとして眺めるだけでなく、**実生活のなかで、自分でもやる**
  （＝好みであり、かつ実践者）。
- ＝**見かけ（筋骨隆々のガテン系の兄貴）と実情（少女趣味の乙女）**の乖離。この人物は
  **魔女／筋肉／殺しの噂／口調**——**全項目で見かけと実情がずれる**、世界の頻出テーマの結晶。

### 土地によって、ずれ方が変わる

- **北部（フスク＝地元）** … 男女差がそこまできつくない土地
  （→ [world/地名と民族.md](../world/地名と民族.md)「男女差の強さも南北で違う」）。
  ゆえに筋肉と兄貴口調は通り、**街の顔役として受け入れられている**。
  ただし**乙女趣味のほうがはみ出す**（刺繍を客に難じられた → 下記）。
- **南部（ペルキ）** … 男女差が強い土地。そこでは彼女は
  **「男装している変人」扱い**になる。
  - **根拠は筋肉ではなく、花（解決）**：**南部では花のモチーフは男性が使う意匠**
    （→ [world/組紐文化.md](../world/組紐文化.md)「花のモチーフは男のもの」）。
    彼女の普段着＝**季節の花を刺繍したカントリードレス**は、南部の目には**男物**に映る。
  - ＝**本人が乙女趣味の極みとして着ているその一着が、そのまま「男装」の証拠になる。**
    可憐であろうとするほど、南では男に近づいてしまう。
- ＝**北では「刺繍の似合わない兄貴」、南では「男装の変人」**。**どちらの土地でも実像（少女趣味の乙女）は
  通らない**——ずれの中身が土地によって入れ替わるだけ。
- **整合（解決）**：セブンズ命名の場面の『**そうね。私たちはセブンズね。**』が女言葉なのは、
  **演出ではなく素の口調**（プライベートの「私」）だから（→ 下記「ベルタとの関係」）。

## 呼称 — 「クラウディ」と呼ばれるのを極端に嫌う

- **名の「クラウディ」で呼ばれるのを極端に嫌う。** 通す呼び名は姓の「**クラウディア**」。
  ＝一般には親称になるはずの**下の名前のほうが地雷**という逆転。
- **理由（確定）＝「クラウディ」が「曇り」だから。それだけ。**
  ＝**過去の出来事や因縁があるわけではない**。単純に、**名の意味が気に入らない**というだけの話。
  - 少女趣味の乙女（→ 上記「中身」）にしてみれば、**自分の名が「曇り」ではあんまり可愛くない**、という感覚。
- **例外的に、ベルタがからかって「クラウディ」と呼ぶことが偶にある**（→ 下記「ベルタとの関係」）。
  ＝地雷を踏めるのは仲がいい相手だけ、という距離の指標にもなっている。

## 魔力ゼロ（最重要）

- **魔力ゼロ。生得能力なし。基本能力（習得魔法）なし。**
- **それでも彼女は正真正銘の魔女である。** **魔女の定義＝不死能力を持つ者＝ガイヤに罰せられた者**であり、
  **魔力の有無は条件に含まれない**（→ [world/魔法.md](../world/魔法.md)「魔女の定義」）。
  ＝彼女は**神罰（スティグマ＋不死）を受けたことによって魔女になった**。
- **自警団のトップに立っていたのは、純粋な筋力だけ。** 魔法は一切使っていない（使えない）。
- **敬虔なティアファ教徒（旧約）。** ガイヤの最後のアナウンス
  （「**間違っても与えた神の力を無益に使ってはいけません**」→ [world/魔法.md](../world/魔法.md)）を
  **声として聞き、その戒めを守って魔法を使っていない**。——**そもそも使えないのだが。**
- **本編の「俺には魔力ってやつはわからない」は文字どおりだった**（→ 下記「本編での様子」）。
  湯の魔力を感じ取れないのではなく、**最初から魔力が無い**。＝旧「要確認」の解決。

### 🔒【伏せ設定】魔力ゼロの理由 — サオージの加護

> **本編では明かされない。読者にも伏せられる秘密**（作者言）。
> **作中の誰も知らない——ガイヤ本人すら知らない。**
> ＝地の文でも登場人物の口からも出さない。資料としてのみ保持する。

- **クラウディアには、サオージの加護が付与されている。**
  （サオージ＝原初の男性神・豊穣神。**世界を去った**とされる神
  → [world/世界の成り立ち.md](../world/世界の成り立ち.md)）。
- **経緯**：**筋肉フェチのサオージが、クラウディアの肉体に惚れて祝福した。**
- **効果（判明分）**：そのため**ガイヤ由来の魔力がゼロ**。
  ＝「**女性は全員ガイヤの加護で魔力を帯びる**」（→ [world/魔法.md](../world/魔法.md)「女性と魔力」）の
  **唯一の例外がここで説明される**。彼女は**加護を持たないのではなく、別の神の加護を持っている**。
- **本人も気づいていない。** クラウディアは自分が加護を受けているなどとは思っておらず、
  **「ちゃんと筋トレしたら、みんなこうなる」**と本気で信じている。
  **「魔法とか使わずにダンベル上げろ」**が持論。
  - ＝**裏表のない人物**（→ 上記「通り名」）ゆえ、これは謙遜でも誤魔化しでもなく**本心からの弁**。
  - ＝**三重に伏せられている**：**作中の誰も知らない／ガイヤも気づいていない／本人すら気づいていない。**
  - **副次効果**：本人が堂々と筋トレ理論を語るので、**周りが加護を疑う余地もない**
    （＝伏せ設定が作中で保たれる仕掛けにもなっている）。
  - **皮肉**：**神の加護持ちが「努力すれば誰でもなれる」と言っている**——才能ある者の無自覚な助言の典型。
    しかもその加護の中身は**魔力を奪うもの**なので、**「魔法を使わずダンベルを上げろ」は、
    彼女自身がそうするしかない条件そのもの**でもある。
- **⚠ここが全部ひっくり返る**：
  - 彼女が**魔法を使えない**のも、**「俺には魔力ってやつはわからない」**のも、すべてこの加護の帰結。
  - そして**ガイヤはそれを知らないまま**、「目立つ＝魔法でイキってる」と決めつけて彼女を罰した。
    ＝**女神は、相方の神が祝福した女を、そうと知らずに罰した**ことになる。
  - **鍛え上げた肉体がサオージの目に留まり → 加護がついて魔力ゼロになり → それでも目立ったから罰された。**
    ＝**筋トレが、加護と神罰の両方を呼び込んだ。**
- **要確認**：加護の効果は「ガイヤ由来の魔力ゼロ」以外にもあるのか（筋力・頑健さへの作用など）／
  祝福の時期（自警団で体を作った後、神罰の前と読めるが未確定）／サオージは去ったはずなのに
  どう祝福したのか（＝サオージ退場の実態に関わる → [canon-notes.md](../canon-notes.md)）／
  この加護と**不死**（ガイヤの神罰）が同居していることの扱い。

## 信仰 — 旧約ティアファ／サオジを持たない

- **信仰は旧約ティアファ**（→ [world/世界の成り立ち.md](../world/世界の成り立ち.md)）。敬虔な信徒で、
  ガイヤの戒めを守って魔法を使わない（→ 上記「魔力ゼロ」）。

  **夫を埋葬するとき、自分のサオジを一緒に埋めてしまった**から
  （→ [world/通貨.md](../world/通貨.md)「サオジと一生」）。
- **理由（語り部の見立て）**：**それだけ夫を愛していた。夫が寂しいだろうと思った**から。
  ＝信仰上の逡巡や自罰ではなく、**まっすぐな情から出た行い**。
- **重み**：サオジは**その人が生涯に結んだ和合の記録＝人生の年輪**であり、**死んだら本人と一緒に埋葬される**もの
  （北部の慣習 → [world/通貨.md](../world/通貨.md)「サオジの死後」）。
  ＝彼女は**自分の人生の記録を、まるごと夫の墓に納めた**ことになる。
  しかも彼女は**不死**（→ 上記「魔力ゼロ」＝神罰）——**自分が埋葬される日は来ない**のだから、
  いずれ自分と共に埋められるはずだったものを、先に手放したことになる。
- **咎めにはならない**：**旧約は「ルールが無いのがルール」と言えるほど戒律を持たない**
  （→ [world/世界の成り立ち.md](../world/世界の成り立ち.md)「戒律の不在」）。
  ＝サオジを持たないことは**戒律違反ではなく、責められる筋合いもない**。

## 刺繍をやめた理由

- **もともとは、ドレスの刺繍を本人が刺していた**（少女趣味の乙女という中身のままの手仕事 → 上記「中身」）。
- だが**洋菓子店の客に「刺繍がかわいすぎる、似合ってないんじゃないか」と言われて傷ついた**。
- **それ以来、刺繍は孫トゥインクル・クラウディアにしてもらっている**（→ 下記「家族」）。
