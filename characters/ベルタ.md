# ベルタ・グラトニー

本編『白い湯の秘密』（→ [stories](../stories/)）に登場する魔女。**セブンズ（10人の魔女の集まり）の一員**
——ただし**「7」のカウント外**（→ [world/セブンズ.md](../world/セブンズ.md)）。呼び名は主に「ベルタ」。

## 信仰 — 無神論者

- **神の声など信じていない無神論者**。ガイヤの最後のアナウンス（全世界の民の意識へ直接届いた声
  → [world/魔法.md](../world/魔法.md)）すら真に受けなかった。
- ＝**神罰を受けて不死になった魔女でありながら、その神を信じていない**。
- **対比**：同じくセブンズのカウント外である**クラウディアは敬虔なティアファ教徒**で、
  声を聞いて戒めを守っていた（→ [クラウディ・クラウディア.md](クラウディ・クラウディア.md)）。
  ＝**信じなかった者と、信じて従った者が、そろって同じ「カウント外」に置かれている。**
- ※関連：晩年の教祖アルバの秘儀（洗礼）を断って自国へ帰った件（下記）も、この不信心と地続きか要確認。

## 立場・身分

- **公爵夫人**という高い身分。その身分ゆえに **MoM（マザーズ・オブ・マザーズ）** に加入した。
- ただし**晩年の教祖アルバの「秘儀」（洗礼＝ワクチン持ちになる儀式／実はレズセックス）を断って
  自国へ帰っている**＝**“名ばかりMoM”**（ワクチンは非保持、アルバの実権から離れた）。
  → [world/マザーズ.md](../world/マザーズ.md)・[canon-notes.md](../canon-notes.md)。
- **派閥「ベルタ派」の名の由来**だが、**本人は所属歴なし・ほぼ無関係**。指向はむしろ
  **クラリス派寄り**（＝名と実の乖離。名だけ看板にされた口 → [world/マザーズ.md](../world/マザーズ.md)）。

## 魔法

### 習得の経緯 — 神罰のあとで、ごく短時間に

- **罰される前**：**ガチればすぐ使えた**が、**やらなかった**。才能の問題ではなく関心の問題
  （無神論者ゆえ神の力の話を真に受けていない → 上記「信仰」）。
- **罰されたあと**：**ガイヤと会ってから魔法に興味を持ち、ほぼ全魔法をごく短時間で習得した。**
  ＝「生得能力以外のほぼすべての魔法をカンスト近くまで」（→ 下記「万能の魔女という看板」）は、
  **神罰の後に、短期間で**成し遂げたもの。
- **皮肉**：ガイヤは「**魔法を悪用した**」という名目で彼女を罰したが、**彼女が魔法を使い始めたのは
  罰されてから**。＝**罰が、咎のほうを後から作った。**
- ※要確認：ガイヤとの対面が興味の引き金になった中身（神の実在を突きつけられたからか、単に力そのものへの
  好奇心か）。「ごく短時間」がどれくらいか。

### 生得能力＝「意識の転移」（ハズレ）

- **魔力で防御されていない人間の意識を、視界内の任意の物体へ移せる。それだけ。**
- **戦闘では全く役に立たない**：女性は全員ガイヤの加護で無意識に魔力を帯び自衛しているため効かない
  （→ [world/魔法.md](../world/魔法.md)「女性と魔力」）。＝実質、効く相手は魔力に守られない**男性**に限られる。
- **他の魔女はこの能力を知らない**（ベルタが伏せている）。

### 「万能の魔女」という看板と、その実像

- **対外的には力は万能「なんでも器用にこなせる」とされ、本人もそう称している。**
- だが実際の凄さは**生得能力ではなく、生まれ持った“人間としての才能”**による：
  - **オールジャンルのオリンピアン並みの身体能力**、
  - **生得能力以外の、ほぼすべての魔法をカンスト近くまで**使いこなす習得センス
    （＝**神罰の後、ごく短時間で**到達 → 上記「習得の経緯」）。
- ＝**見かけ（万能の天才魔女）と実情（生得はハズレ、凄さは人間的才能）の乖離**。

### 本編での主な超常（すべて基本能力側）

- **大気圏再突入からの生還** … 基本系統の**「肉体強化」**で、物理でそのまま突っ込んだだけ。
- **目から出す「怪光線」** … **「炎の魔法」**の応用。
- ＝どちらも生得能力ではなく**基本能力**（→ [world/魔法.md](../world/魔法.md)「派手さ ≠ 生得能力」）。

## 「7」に数えられていない理由（解決）

- セブンズの「7」＝**魔法を使っていて罰せられた者**の頭数（→ [world/セブンズ.md](../world/セブンズ.md)「名の由来」）。
- **ベルタは、ガイヤが罰しに来た時点では魔法を使っていなかった。** ゆえに7に入らない。
- **理由＝無神論者だったから。** **神の声など信じていなかった**——ガイヤの最後のアナウンス
  （＝魔法への目覚めの引き金 → [world/魔法.md](../world/魔法.md)）を真に受けなかった。
- **ただし「無実」ではない**：**その時点で既にキルカウントは二桁**（間接的にではあるが）。
  ＝**罰せられるべき人物ではあった。少なくとも魔法は使っていない、というだけ**
  （→ [world/キルカウント.md](../world/キルカウント.md)）。
- **クラウディアとの対比**：カウント外の2人はどちらも「魔法を使わずに罰せられた」組だが、中身は正反対。
  - **クラウディア** … 魔力ゼロ・キルカウント0・敬虔な信徒＝**完全な無実**（冤罪）。
  - **ベルタ** … 魔法は使えるが未使用・キルカウント二桁・**無神論者**＝**咎はあるが、罰の名目とは別の咎**。
  - ＝ガイヤは「魔法を悪用した者」を罰したつもりで、**この2人については名目を完全に外している**。
    にもかかわらず、**ベルタに関しては結果として「罰されるべき者」に当たっていた**（理由は違えど）。
- **②③解決＝使えなかったのではなく、やる気がなかっただけ**：**ベルタはガチれば（本気を出せば）すぐ魔法を使えた。**
  才能は最初からあり、**関心がなかった**（無神論ゆえ神の力の話を真に受けなかった）だけ。
  そして**ガイヤと会ってから魔法に興味を持ち、ほぼ全魔法をごく短時間で習得した**（→ 下記「魔法の習得」）。
- **要確認**：**「間接的に」の中身**（何をして二桁に至ったのか。公爵夫人という身分・立場が絡むのか）。

## 要確認
